How Smart Focus works

Each pending task gets a score based on 3 factors. The highest scored task is ranked first.

Score = (Priority × 3) + (Overdue Days × 2) + (Age × 0.5)
Priority High = 9pts  ·  Medium = 6pts  ·  Low = 3pts
Overdue Each day past due date adds 2pts (3 days overdue = +6pts)
Age Each day since created adds 0.5pts (capped at 5pts so old tasks don't dominate)

Example: High priority task, 3 days overdue, created 8 days ago → 9 + 6 + 4 = 19 pts

Tasks ranked by urgency Score = Priority + Overdue Days + Age
